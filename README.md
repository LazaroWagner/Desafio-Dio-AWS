# RELATÓRIO DE IMPLEMENTAÇÃO DE SERVIÇOS AWS
Data: 13/03/2026
Empresa: Doces Mar
Responsável: Lazaro Marinho

## Introdução
Este relatório apresenta o processo de implementação de ferramentas na empresa Doces Mar, realizado por Lazaro Marinho.
O objetivo do projeto foi elencar 3 serviços AWS com foco em redução de custos imediatos e melhoria na gestão de vendas de doces.

## Descrição do Projeto
O projeto foi dividido em três etapas, cada uma com objetivos específicos:

### Etapa 1

•	Amazon S3

•	Foco: Armazenamento de imagens e catálogos de produtos.

•	Caso de uso: Guardar fotos dos doces e cardápios digitais, permitindo acesso rápido e seguro sem necessidade de servidores locais.

### Etapa 2

•	Amazon RDS (MySQL)

•	Foco: Banco de dados gerenciado.

•	Caso de uso: Centralizar informações de clientes, pedidos e estoque em um banco de dados escalável, sem custos de manutenção de servidores físicos.

### Etapa 3

•	AWS CloudFront

•	Foco: Distribuição de conteúdo.

•	Caso de uso: Garantir que o site de vendas online carregue rapidamente para clientes em qualquer região, reduzindo latência e melhorando a experiência de compra.


## Conclusão

A implementação dos serviços na empresa Doces Mar trouxe benefícios como:

•	Redução de custos com infraestrutura local.

•	Maior agilidade na gestão de pedidos e estoque.

•	Melhor experiência para os clientes com acesso rápido ao catálogo online.

Recomenda-se a continuidade da utilização dos serviços implementados e a avaliação de novas soluções AWS, como Amazon Pinpoint para campanhas de marketing e fidelização de clientes.
